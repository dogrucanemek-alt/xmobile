Gizlilik Politikası

Son güncelleme: 17 Mayıs 2026

1. Toplanan Veriler

xmobile yalnızca cihazınızda saklanan aşağıdaki verileri toplar:

  • Kıyafet fotoğrafları ve isimleri
  • Profil bilgileri (ten rengi, saç rengi, göz rengi, boy, kilo, cinsiyet)
  • Kombin geçmişi
  • Konum bilgisi (yalnızca anlık hava durumu için, kaydedilmez)

Kıyafet fotoğrafları, profil bilgileri ve kombin geçmişi cihazınızda yerel olarak saklanır. Giriş yapan kullanıcılar için kimlik doğrulama bilgileri ve Jarvis AI asistan geçmişi güvenli bulut altyapısında tutulur.

2. Üçüncü Taraf Hizmetler

Uygulama aşağıdaki üçüncü taraf hizmetleri kullanmaktadır:

  • Anthropic Claude API — Kombin önerileri ve kıyafet tanıma için fotoğraflar gönderilir. (anthropic.com/privacy)
  • OpenWeatherMap API — Anlık hava durumu verisi için konumunuz gönderilir. (openweathermap.org/privacy-policy)
  • Meshy API — 3D model oluşturmak için kıyafet adı gönderilir. (meshy.ai/privacy)
  • Fashn.ai API — Sanal deneme için kıyafet ve model fotoğrafları gönderilir. (fashn.ai/privacy)
  • Replicate API — Arka plan temizleme için fotoğraf gönderilir. (replicate.com/privacy)
  • OpenAI API — Genel asistan sorguları için metin/görsel gönderilir. (openai.com/privacy)
  • Supabase — Kimlik doğrulama ve veri saklama. (supabase.com/privacy)
  • Sentry — Anonim hata raporları. (sentry.io/privacy)
  • RevenueCat — Abonelik yönetimi. (revenuecat.com/privacy)
  • Mixpanel — Anonim kullanım analitiği. (mixpanel.com/privacy)

3. Veri Güvenliği

Kıyafet fotoğraflarınız ve profil bilgileriniz cihazınızın yerel depolama alanında (AsyncStorage ve dosya sistemi) tutulur.

Giriş yapan kullanıcılar için kimlik doğrulama bilgileri ve Jarvis AI asistan geçmişi şifreli bağlantı (HTTPS/TLS) üzerinden Supabase bulut altyapısında güvenli biçimde saklanır.

API anahtarlarınız uygulama paketinde yer almaz; tüm AI ve harici servis çağrıları güvenli bir sunucu proxy üzerinden gerçekleştirilir.

4. KVKK ve GDPR Uyumu

6698 sayılı Kişisel Verilerin Korunması Kanunu (KVKK) ve GDPR kapsamında verileriniz üçüncü taraflarla satılmamakta veya kiralanmamaktadır. Yalnızca yukarıda belirtilen üçüncü taraf API hizmetlerine ilgili veriler aktarılmaktadır.

İstediğiniz zaman uygulamadaki "Hesap Sil" özelliği ile tüm verilerinizi silebilir veya dogrucanemek@gmail.com adresine yazarak verilerinizin silinmesini talep edebilirsiniz.

5. Çocukların Gizliliği

xmobile 13 yaşın altındaki çocuklara yönelik değildir. Bilerek 13 yaş altı kişilere ait veri toplamıyoruz.

6. Değişiklikler

Bu politika zaman zaman güncellenebilir. Önemli değişiklikler uygulama içi bildirim veya e-posta ile duyurulur. Değişiklikler bu sayfada yayınlandığı tarihten itibaren geçerlidir.

Privacy Policy

Last updated: May 17, 2026

1. Data We Collect

xmobile collects only the following data stored locally on your device:

  • Clothing photos and names
  • Profile information (skin tone, hair color, eye color, height, weight, gender)
  • Outfit history
  • Location (only for instant weather data, never stored)

Clothing photos, profile data, and outfit history are stored locally on your device. For signed-in users, authentication and Jarvis AI history are stored on secure cloud infrastructure.

2. Third-Party Services

The app uses the following third-party services:

  • Anthropic Claude API — Photos for outfit suggestions and clothing recognition. (anthropic.com/privacy)
  • OpenWeatherMap API — Location for real-time weather. (openweathermap.org/privacy-policy)
  • Meshy API — Clothing names to generate 3D models. (meshy.ai/privacy)
  • Fashn.ai API — Clothing and model photos for virtual try-on. (fashn.ai/privacy)
  • Replicate API — Photos sent for background removal. (replicate.com/privacy)
  • OpenAI API — Text/image for general assistant queries. (openai.com/privacy)
  • Supabase — Authentication and data storage. (supabase.com/privacy)
  • Sentry — Anonymous crash reports. (sentry.io/privacy)
  • RevenueCat — Subscription management. (revenuecat.com/privacy)
  • Mixpanel — Anonymous usage analytics. (mixpanel.com/privacy)

3. Data Security

Your clothing photos and profile data are stored locally on your device (AsyncStorage and file system).

For signed-in users, authentication and Jarvis AI history are securely stored on Supabase cloud infrastructure over encrypted connections (HTTPS/TLS).

API keys are not bundled in the app; all AI and external service calls are routed through a secure server-side proxy.

4. GDPR and KVKK Compliance

Under GDPR and Turkish KVKK Law No. 6698, your data is not sold or rented to any third parties. Data is only shared with the listed third-party APIs to provide app functionality.

You may delete all your data at any time via the in-app "Delete Account" feature, or by emailing dogrucanemek@gmail.com to request deletion.

5. Children's Privacy

xmobile is not directed at children under 13. We do not knowingly collect personal information from children under 13.

6. Changes

This policy may be updated from time to time. Material changes are announced via in-app notification or email. Changes take effect from the date posted on this page.

7. Contact

For questions about privacy, GDPR, KVKK, or data deletion requests:
dogrucanemek@gmail.com
