Değişiklikleri akıllı commit mesajıyla kaydet ve GitHub'a push et.

Adımlar:
1. `git status` ve `git diff` ile değişiklikleri incele
2. Değişikliklere göre Türkçe kısa bir commit mesajı oluştur (örn: "Avatar renk düzeltildi", "Kombin prompt sadeleştirildi")
3. Proje dosyalarını stage'e al, `.claude/`, `expo_out.txt`, `expo_err.txt`, `qr_expo.png` hariç tut
4. `git commit` ve `git push` yap, Co-Authored-By satırını ekle
5. Commit hash ve mesajı göster
